title: Attention weights
description: A minimal, real self-attention computation, stepped through the causal mask and visualized as a heatmap.
---

Self-attention lets every token look at every other token and decide, per pair, how much
to pay attention to it. Here we compute it for real, on a 6-token toy sentence, and watch
the causal mask turn "every token sees everything" into "every token sees only what came
before it."

## Setup

```python cell
import math
import torch

torch.manual_seed(7)

tokens = ["The", "cat", "sat", "on", "the", "mat"]
d_model = 8
n = len(tokens)
print(f"{n} tokens, d_model={d_model}")
```

Random (but seeded, for reproducible snapshots) token embeddings and query/key/value projections:

```python cell
X = torch.randn(n, d_model) * 0.3
W_q = torch.randn(d_model, d_model) * 0.3
W_k = torch.randn(d_model, d_model) * 0.3
W_v = torch.randn(d_model, d_model) * 0.3

Q = X @ W_q
K = X @ W_k
V = X @ W_v
print("Q, K, V shape:", tuple(Q.shape))
```

## Raw scores

Scaled dot-product attention scores — how much each query token resonates with each key token,
before any masking:

```python cell
raw_scores = (Q @ K.T) / math.sqrt(d_model)
print(raw_scores.round(decimals=2))
```

## The causal mask

A language model shouldn't let a token attend to positions after it. We build a boolean mask
for the upper triangle (excluding the diagonal) and fill those positions with $-\infty$ before
the softmax, so they vanish (become exactly 0) once normalized:

```python cell
mask = torch.triu(torch.ones(n, n), diagonal=1).bool()
masked_scores = raw_scores.masked_fill(mask, float("-inf"))
print(masked_scores.round(decimals=2))
```

## Softmax

$$
\text{softmax}(x)_i = \frac{e^{x_i}}{\sum_j e^{x_j}}
$$

```python cell
weights = torch.softmax(masked_scores, dim=-1)
print(weights.round(decimals=2))
print("row sums:", weights.sum(dim=-1).round(decimals=4))
```

A quick matplotlib sanity check of the same matrix — a static counterpart to the interactive
heatmap below:

```python cell
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(4, 4))
im = ax.imshow(weights.numpy(), cmap="Blues")
ax.set_xticks(range(n), tokens, rotation=45, ha="right")
ax.set_yticks(range(n), tokens)
ax.set_title("Attention weights")
fig.colorbar(im, fraction=0.046, pad=0.04)
fig.tight_layout()
plt.show()
```
